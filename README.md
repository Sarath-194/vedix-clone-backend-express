#vedix-clone.
<h2>VEDIX-Clone</h2>

<p>Clone the Website <a href='https://vedix.com/'>https://vedix.com</a></p>
<p>Vedix is a beauty seller in Online for various Skin and Hair issues. Vedix have Quizs to Analyse the Issue regard to Skin/Hair and offers Products for Better Results.</p>
<p> We are Team of 3 Developers to clone The vedix with Back-end. Vedix-clone is Developed with combination of Back-end and Front-end technology for front-end we have used EJS, CSS, and Advanced JS and for Back-end Mongoose, Express, MongoDB Atlas, and Node.js . Focused more on Enriching same features from the Actual website.</p>

<p> Deploy Link: <a href='https://vedix-clone-backend.herokuapp.com/'>https://vedix-clone-backend.herokuapp.com</a> </p>

<h2> Tech-Stack</h2>
<p> We have Front-end and Back-end technologies like</p>
<li> EJS </li>
<li> CSS </li>
<li> Advanced JS</li>
<li> Mongoose </li>
<li> Express </li>
<li> Node.js</li>
<li> Mongo DB Atlas</li>
<li> Heroku(for Deploy)</li>


<h2> Libraries</h2>
<p> For Icons and Some Font we have used Font Awesome <a href='https://fontawesome.com/'>https://fontawesome.com/</a></p>
